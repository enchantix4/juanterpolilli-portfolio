# Quick Start Guide

## To View Your Website

### Option 1: Install Node.js (Recommended)

1. **Install Node.js:**
   - Visit: https://nodejs.org/
   - Download and install the LTS version
   - Or use Homebrew: `brew install node`

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Open in browser:**
   - The server will start at: http://localhost:3000
   - Cursor may automatically open it, or you can click the link in the terminal

### Option 2: Use Cursor's Preview (if available)

- Cursor may have a built-in preview feature
- Look for a "Preview" or "Open in Browser" button in Cursor's interface

## Troubleshooting

If you see errors:
1. Make sure Node.js is installed: `node --version`
2. Make sure you're in the project directory
3. Try deleting `node_modules` and running `npm install` again
